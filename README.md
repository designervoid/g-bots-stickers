# G-Bots Stickers

## Prerequisites

`npm16.13.2`, `npm 8.1.2`

Installing packages:

```zsh
npm install
```

## Generate sticker chunk reverse

```zsh
npm run build:generate-sticker-chunk-reverse -- --outputVideoType webm --sourceDir sources/mp4 --bitRate 2500 --size 512:512
```
